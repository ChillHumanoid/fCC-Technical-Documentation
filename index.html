Introduction

Python is a popular programming language used in dozens of careers fields. It was created by Guido van Rossum, and released in 1991. The newest release, Python 3.8, was released in September of 2022.

It is used for:

  • Web Development (server-side),
  • Software Development,
  • Mathematics and other similar equations,
  • System Scripting,
  • Data Analysis,
  • and MUCH MORE!

Hello World

Python was designed for readability, and has some similarities to the English language with influence from mathematics. With a little bit of time and practice, reading Python code should become fairly easy.

Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses. Look at the code below and think about the similarities and differences between Python and other programming languages such as HTML or CSS.

Example 1:

print("Hello World!")

Example 2:

if 5 > 2 // print("Five is greater than two!")
Variables

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Example 1:

x = 5 // print(x); this code would print 5.

Example 2:

y = "John" // print(y); this code would print John.

Example 3:

z = "Ranger" // print(z); this code would print Ranger.
